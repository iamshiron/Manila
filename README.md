# Manila
Manila is a build system for C++ using JavaScript for Build Scripts (maybe TypeScript support in the future).

## Project Structure
- **Manila/**
The main sources of the build system

- **Manila.API/**
Contains the binding between the Build System and its scripting API

- **Manila.CLI/**
The executeable that will get shipped and runs the build system from command line

- **run/**
A test directory that is used in Debug to test the features of the build system
