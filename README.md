# Manila
Manila is a build system for C++ using JavaScript for Build Scripts (maybe TypeScript support in the future).

## Project Structure
- **Manila/**
The main sources and core of the build system

- **Manila.API/**
Contains the bindings between a Manila Plugin and the Manila Core.

- **Manila.CPP/**
An example plugin containing the required sources for the C++ support

- **run/**
A test directory that is used in Debug to test the features of the build system
